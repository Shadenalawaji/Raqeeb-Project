# Raqeeb-Project
This project presents an AI-based system for detecting road obstacles using real-time data from a vehicle-mounted camera. The system aims to alert drivers to potential hazards and enhance road safety while supporting smart infrastructure management.

## Overview
Raqeeb is an AI-powered road monitoring system designed to detect and classify road obstacles in real time using computer vision and deep learning techniques. The system connects to a dashcam feed, analyzes live road conditions, and provides instant alerts to drivers while also notifying authorities for high-risk hazards.

The project aims to enhance road safety, reduce accidents, and support smart city infrastructure through real-time intelligent detection.

---

## Objectives
- Detect road obstacles in real time from dashcam footage
- Classify hazards based on severity
- Provide immediate alerts to drivers
- Notify municipalities for critical road issues
- Build a dataset for road hazard pattern analysis

---

## Key Features
- Real-time obstacle detection using AI
- Road hazard classification by severity level
- Dashcam integration
- Instant driver alerts through mobile app or SMS
- Multi-level notification system
- Data collection for pattern and trend analysis
- Supports smart city infrastructure use cases

---

## Technologies Used
- Artificial Intelligence and Deep Learning
- Computer Vision
- Real-time image analysis
- Python / TensorFlow / PyTorch / OpenCV
- Flutter / Android / iOS
- Firebase / MySQL / Backend API

---

## System Workflow
1. Capture real-time video from dashcam
2. AI model analyzes frames
3. Detect and classify road obstacles
4. Assign severity level
5. Send alert to driver
6. Forward critical cases to municipality system
7. Store data for analytics

---

## Hazard Types Detected
- Potholes
- Road bumps
- Water puddles
- Surface damage
- Unexpected obstacles

---

## Alert Mechanism
- Driver receives instant notification via mobile application or SMS
- High-risk hazards are automatically reported to authorities
- Escalation is based on severity level
